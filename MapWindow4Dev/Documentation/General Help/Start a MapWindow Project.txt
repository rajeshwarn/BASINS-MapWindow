<b>Steps to Open a New MapWindow Project</b>
<ol>
<li>Double click on the MapWindow icon that is on your desktop (or click Start->All Programs (or Programs)->MapWindow GIS->MapWindow GIS). A welcome screen appears titled Welcome to MapWindow.
<li>Click on the hyperlink titled Add Data to This Project.
</ol>
<b>Steps to Open an Existing MapWindow Project</b>
<ol>
<li>Double click on the MapWindow icon that is on your desktop (or click Start->All Programs (or Programs)->MapWindow GIS->MapWindow GIS). A welcome screen appears titled Welcome to MapWindow.
<li>Click on the hyperlink titled Open Document. A dialog box appears and allows you to add a MapWindow Project file.
<li>Search for an existing MapWindow project file and highlight it by clicking on the file name.
<li>Click the Open button. <i>(Note: to learn more about MapWindow Project Files, click <a href="Menus\File">here</a> and then browse for the title To Save a MapWindow Project)</i></ol>
