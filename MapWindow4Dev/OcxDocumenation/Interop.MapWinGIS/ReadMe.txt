// To update class members after new items where added:
// 1) Rename the class, for example IShapefile -> IShapefile1
// 2) Make it implement base interface from MapWinGIS, for example 'class IShapefile1: IShapefile'
// 3) Choose 'Implement interface' from the smart tag; new members will added
// 4) Rename the class back and remove base interface
